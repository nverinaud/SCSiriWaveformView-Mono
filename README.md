# SCSiriWaveformView Mono

Mono port of the excellent [SCSiriWaveformView library](https://github.com/stefanceriu/SCSiriWaveformView).

## SCSiriWaveformView

UIView subclass that reproduces the waveform effect seen in Siri on iOS 7.
